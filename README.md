
# Modern React & Tailwind Portfolio Website

A sleek, modern, and fully responsive personal portfolio website built with React, Vite, and Tailwind CSS. This project showcases skills, experience, projects, and insights through a clean, animated, and interactive user interface.

**[Live Demo](https://your-live-demo-link.com) (replace with your link)**

---

## ✨ Features

- **Blazing Fast:** Built with Vite for near-instant server start and Hot Module Replacement (HMR).
- **Modern Tech Stack:** Utilizes React, TypeScript, and Tailwind CSS for a robust and maintainable codebase.
- **Fully Responsive:** Looks great on all devices, from mobile phones to desktops.
- **Smooth Animations:** Subtle and performant animations using Framer Motion to enhance user experience.
- **Interactive UI:** Dynamic components for filtering projects, viewing testimonials, and reading blog posts.
- **Component-Based Architecture:** Clean and organized code structure with reusable components.
- **SEO Friendly:** Structured with SEO best practices in mind.
- **Easy to Customize:** Easily update your personal information, projects, and blog posts by editing simple data files.

## 🚀 Getting Started

### Running the Development Server

To start the local development server, run the following command:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) (or the address shown in your terminal) in your browser to see the result. The app will automatically reload if you change any of the source files.

### Building for Production

To create a production-ready build of the application, run:

```bash
npm run build
```

This will create a `dist` folder in the project root with the optimized and minified files for deployment.

## 📂 Project Structure

The project follows a standard React application structure:

```
modern-portfolio-website/
├── public/                # Static assets (images, fonts)
├── src/
│   ├── components/        # Reusable React components
│   ├── data/              # Static data for portfolio, blog, etc.
│   ├── hooks/             # Custom React hooks
│   ├── types/             # TypeScript type definitions
│   ├── App.tsx            # Main application component
│   ├── main.tsx           # Entry point of the application
│   └── index.css          # Global styles and Tailwind imports
├── .eslintrc.cjs          # ESLint configuration
├── tailwind.config.js     # Tailwind CSS configuration
├── vite.config.ts         # Vite configuration
└── package.json           # Project dependencies and scripts
```
